キーボードショートカット

グリッド表示・フルスクリーン表示で使用できるキーボードショートカットの一覧です。

グリッド表示

ナビゲーション

キー操作
← → ↑ ↓選択を移動
Enter選択中の画像をフルスクリーンで開く / フォルダに移動
Home先頭へ移動
End末尾へ移動
Page Up1ページ上にスクロール
Page Down1ページ下にスクロール
BS親フォルダへ移動
Ctrl+↑フォルダツリーで前のフォルダへ
Ctrl+↓フォルダツリーで次のフォルダへ

選択・チェック

キー操作
Space選択中のアイテムのチェックを ON/OFF
Shift+矢印キー範囲チェック（移動元〜移動先を一括チェック）
Ctrl+クリックアイテムのチェックを ON/OFF

ファイル操作

キー操作
Ctrl+C選択/チェック済みファイルをコピー
Ctrl+X選択/チェック済みファイルをカット
Ctrl+Vクリップボードのファイルをペースト
Delete選択/チェック済みファイルをごみ箱へ移動
R選択画像を右に回転
L選択画像を左に回転

レーティング

キー操作
F1〜F5選択/チェック済み画像に★1〜★5 のレーティングを付与（ZIP 内画像・PDF ページも対象）
F6レーティングを解除（0 に戻す）
Ctrl+A現在のフィルタで表示中の画像を全てチェック

その他

キー操作
Ctrl+Oフォルダを開くダイアログ
Ctrl+Fメタデータ検索バーを表示
Ctrl+Sお気に入り検索バーを表示（フォルダ・ZIP・PDF 名の横断検索）
Ctrl+マウスホイール列数を増減
Alt+1〜9列数を 1〜9 に切り替え
Alt+0列数を 10 に切り替え
マウスホイールグリッドをスクロール
右クリックコンテキストメニューを表示

フルスクリーン表示

画像の移動・終了

キー操作
→ / ↓次の画像へ
← / ↑前の画像へ
Homeフォルダ内の最初の画像へ
Endフォルダ内の最後の画像へ
Ctrl+↓フルスクリーンのまま次フォルダの先頭画像へジャンプ
Ctrl+↑フルスクリーンのまま前フォルダの末尾画像へジャンプ
Escグリッド表示に戻る

見開き表示

キー操作
11ページ表示（通常）
2見開き 左→右
3見開き 左→右（表紙あり）
4見開き 右→左
5見開き 右→左（表紙あり）
Shift+矢印キー見開きモード中、1ページ送り
ℹ️
右→左モード（4/5）では ← と → の意味が反転します。 ↑/↓ は常に前/次です。

回転・操作

キー操作
R右に 90° 回転
L左に 90° 回転
F1〜F5表示中の画像に★1〜★5 のレーティングを付与
F6レーティングを解除
E消しゴムモード ON（マスク指定）/ AI 補完実行（詳細）
I / Tabメタデータパネルの固定表示切り替え
Pスライドショー 再生 / 停止
Spaceスライドショー再生中：停止 ／ 停止中：チェック ON/OFF
Enter動画の場合、外部プレイヤーで再生

画像補正

キー操作
UAI アップスケールモデルを順にサイクル切替（現在のページに個別適用）
NAI ノイズ除去を ON / OFF（現在のページに個別適用）
Ctrl+1〜9 / Ctrl+0保存スロット 1〜10 を現在のページに適用（= そのページを個別化）
Ctrl+Backspace現在ページの個別補正設定を解除（標準値に戻す）
ℹ️
補正パネル（画面左端にマウスを寄せると表示）のスライダーを動かすと、自動的にそのページの個別補正として保存されます。 パネル上部の「全画像に適用」「全画像から削除」「標準にする」「個別設定を解除」の 4 ボタンで、適用範囲を明示的に操作できます。 個別設定のあるページを開くと右上に「ページ補正適用」のトーストが表示されます。

マウス操作

操作説明
マウスホイール（上/下）前後の画像に移動
Ctrl+マウスホイールマウス位置を中心にズーム（0.1〜50 倍）
左ドラッグ（ズーム中）パン（表示範囲を移動）
Ctrl+左ドラッグ画像を任意の角度に回転（一時的、保存なし）
ダブルクリックズーム / パン / 回転をリセット
画面の右半分をクリック次の画像へ
画面の左半分をクリック前の画像へ
右クリック（短押し）グリッド表示に戻る
右クリック（長押し）コンテキストメニュー（パスコピー・フォルダを開く・アプリで開く等）
画面右 25% にカーソル移動メタデータパネルをホバー表示
画面上部にカーソル移動ファイル情報バーと操作ボタンを表示

画像分析モード

Z キーまたはホバーバーの 🔬 ボタンで分析モードに入ります。詳しい用途はフルスクリーン表示 > 画像分析モードをご覧ください。

分析モード — 切り替え

キー / 操作説明
Z分析モード ON / OFF
ホバーバー 🔬 ボタン分析モード ON / OFF
パネル右上 × ボタン分析モード OFF

分析モード — ズーム / パン

操作説明
マウスホイールズーム（マウス位置中心、0.1〜50 倍）
左ドラッグ（修飾キーなし）パン（表示範囲を移動）
ダブルクリックズーム / パンをリセット

分析モード — カラー / フィルター

操作説明
マウスホバー（画像上）カーソル位置の色を RGB / HSV / L*a*b* で取得
右クリック比較色を固定（ガイドライン・フィルターも解除）
Gグレースケール表示 ON / OFF
Mモザイクグリッド表示 ON / OFF

分析モード — スケールグリッド（等身計測）

操作説明
Shift + 左ドラッグ赤色のスケールグリッド
Ctrl + 左ドラッグ黒色のスケールグリッド
Alt + 左ドラッグ白色のスケールグリッド
右クリックスケールグリッドを消去

分析モード — 色差強調フィルター

操作倍率
Shift + 右クリック×2
Ctrl + 右クリック×5
Alt + 右クリック×10
Ctrl+Alt + 右クリック×20
右クリック（修飾キーなし）フィルター解除

消しゴムモード

E キーで開始。モード中は通常のフルスクリーンショートカットは無効化され、以下の専用キーのみ有効です。 詳細は 消しゴムモード ページを参照してください。

モード操作・ツール切替

キー操作
E開始 / AI 補完実行（2 回目）
Esc消しゴムモード終了（フルスクリーンは継続、マスクは自動保存）
S選択ツール (ベクタオブジェクトのクリック選択・編集)
B筆ツール
L囲みツール
V縦線ツール
H横線ツール
I直線ツール
D / F描画モード / 消去モードに切替
Ctrl+ZUndo（最大 20 手）
矢印キーマスク全体 (or 選択ベクタ) を 1 px 平行移動 (Ctrl+で 10 px)
[ / ]±0.1° 回転 (Ctrl+で ±1°)
Del選択中のベクタオブジェクトを削除

マスクスロット

保存・ロードは消しゴムモードの左パネルのボタンから行います。 フルスクリーン表示のまま 1 キーで適用したい場合は以下を使います (消しゴムモードに入る必要なし)。

キー操作
F7 / F8フルスクリーン表示中、保存 1 / 保存 2 のマスクを現ページに即適用 (AI 補完まで実行)

マウス操作

操作動作
左ドラッグ選択中ツールでマスク描画／消去
ベクタをクリック (選択ツール時)選択 (黄色の枠が表示)。空領域クリックで解除
選択中にドラッグベクタオブジェクトを平行移動
選択中に Shift+ドラッグ縦＝回転、横＝太さ変更
直線の端点をドラッグ (選択ツール時)始点 / 終点の位置変更
マウスホイール筆・直線ツール時は太さ変更（その他ツール時は無効）
Ctrl+ホイール画像ズーム
Shift+ドラッグ（筆・直線描画中）太さ変更（右／下＝拡大、左／上＝縮小）
Shift+ドラッグ（縦線描画中）左右＝線をパン、上下＝線を回転（傾き調整）
Shift+ドラッグ（横線描画中）上下＝線をパン、左右＝線を回転（傾き調整）
ℹ️
フルスクリーン表示のキーボード操作はウィンドウにフォーカスがある場合のみ有効です。 他のウィンドウをクリックした後は、mImageViewer のウィンドウを一度クリックしてから操作してください。
