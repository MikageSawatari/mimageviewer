対応フォーマット

mImageViewer が対応している画像・動画フォーマットの一覧です。

静止画（内蔵デコーダ）

以下のフォーマットは追加インストール不要で表示できます。

拡張子フォーマット備考

.pngPNGAI 生成メタデータ（tEXt / iTXt / zTXt）対応

静止画（WIC 経由）

Windows イメージング コンポーネント（WIC）を使用して表示します。 Windows 10 / 11 では標準で利用できますが、一部フォーマットは Microsoft Store からコーデックの追加が必要です。

ℹ️
Microsoft Store コーデックのインストール： Microsoft Store で「HEIF 画像拡張機能」「Microsoft Raw 画像拡張機能」などを検索してインストールしてください。 無料で入手できます。インストール後、mImageViewer の再起動は不要です。

動画（サムネイルのみ）

動画ファイルはサムネイル表示のみ対応しています。再生機能はありません。 サムネイルは Windows Shell API（IShellItemImageFactory）を使用して取得します。

拡張子フォーマット
.mp4MPEG-4
.aviAVI
.movQuickTime
.mkvMatroska
.wmvWindows Media Video
.mpg / .mpegMPEG
💡
動画サムネイルが表示されない場合は、対応するコーデックが Windows にインストールされていない可能性があります。 K-Lite Codec Pack などの一般的なコーデックパックをインストールすることで改善する場合があります。
